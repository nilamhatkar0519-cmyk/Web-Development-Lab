Experiment 12: XSLT Implementation

Objective: To learn XSLT and its application for transforming XML to HTML.

Name: Nilam Hatakar | Roll No: 70 
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="cat.xsl"?>
<students>
  <student id="70">
    <name>Nilam Hatakar</name>
    <course>B.Tech CSE</course>
    <semester>IV</semester>
  </student>
</students>
<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:template match="/students">
    <xsl:for-each select="student">
      <div>
        <p>Roll No: <xsl:value-of select="@id"/></p>
        <p>Name: <xsl:value-of select="name"/></p>
        <p>Course: <xsl:value-of select="course"/></p>
        <p>Semester: <xsl:value-of select="semester"/></p>
      </div>
    </xsl:for-each>
  </xsl:template>
</xsl:stylesheet>

Transformed Output:

Experiment 13: Student Application using XSLT

Objective: To apply XSLT and conditional statements for student data transformation.

Name: Nilam Hatakar | Roll No: 70 
<?xml version="1.0" encoding="UTF-8"?>
<students>
    <student id="101">
        <name>Sahil Patil</name>
        <course>BTech</course>
        <marks>95</marks>
    </student>
    <student id="102">
        <name>Nilam Hatkar</name>
        <course>BTech</course>
        <marks>80</marks>
    </student>
</students>
<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0">
  <xsl:template match="/students">
    <xsl:for-each select="student">
      <tr>
        <td><xsl:value-of select="@id"/></td>
        <td><xsl:value-of select="name"/></td>
        <td><xsl:value-of select="course"/></td>
        <td>
          <xsl:choose>
            <xsl:when test="marks >= 85">A</xsl:when>
            <xsl:when test="marks >= 75">B</xsl:when>
            <xsl:otherwise>C</xsl:otherwise>
          </xsl:choose>
        </td>
      </tr>
    </xsl:for-each>
  </xsl:template>
</xsl:stylesheet>

Transformed Output with Conditional Grading:

Roll NoNameCourseGrade
101Sahil PatilBTechA
102Nilam HatkarBTechB
